# レイヤーレコメンデーション機能 

## 概要
ユーザーが入力したキーワードをもとに、端末内でレイヤーとの類似度を計算し、類似している上位5つのレイヤーを自動的に推薦するインテリジェントなインターフェースです。

単なるキーワード検索ではなく、埋め込みベクトルを用いた意味検索（Semantic Search）を行うことで、「防災」「気象」といった抽象的な入力からでも、関連性の高いレイヤーを的確に絞り込みます。
地図利用者が直感的に必要な情報へアクセスできるよう、推薦結果を即座に地図上にオーバーレイ表示できる「地図情報コンシェルジュ」として機能します。

## 導入方法
`Container.svg` に以下の `animation` タグを追記することで、本機能を地図アプリケーションに統合できます。

```xml
<animation xlink:href="./appLayers/layer-recommender/recommender.svg" title="レイヤー推奨" x="-30000" y="-30000" width="60000" height="60000" class="" opacity="0.6" visibility="hidden"/>
```

## 利用上の注意点
導入を検討される方は、以下の技術的なポイントにご留意ください。

- **プライバシーとセキュリティ**: 本機能は入力内容や類似度計算をすべて端末内（ブラウザ上）で完結させています。検索クエリや個人情報がサーバー側に送信されることはありません。
- **パフォーマンス**: 類似度計算はクライアント側の計算資源（CPU/GPU）を使用します。大量のレイヤーを登録する場合、初期化時に一定の処理時間がかかることがあります。
- **モデルの依存関係**: ベクトル化にはHuggingFaceのTransformerモデルを使用しています。初回実行時にモデルデータの読み込み（通信）が発生します。
- **推奨環境**: WebGPU対応ブラウザでの利用を推奨します。環境が適さない場合は自動的にWASM（WebAssembly）環境へフォールバックされます。
