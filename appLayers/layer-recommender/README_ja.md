# レイヤーレコメンデーション機能 

## 概要
ユーザーが入力したテキストから文脈や地理的な意図を理解し、プロジェクト内の膨大な地図レイヤーの中から最適なものを自動的に推薦（レコメンデーション）するためのインテリジェントな検索・可視化インターフェースです。

単なるキーワード検索にとどまらず、埋め込みベクトルを用いた意味検索（Semantic Search）を行うことで、
「防災」「気象」といった抽象的な検索語からでも、関連性の高いレイヤーを的確に絞り込むことができます。
地図利用者が直感的に必要な情報へアクセスできるよう、レコメンデーション結果を即座に地図上にオーバーレイ表示できる「地図情報コンシェルジュ」としての役割を果たします。
